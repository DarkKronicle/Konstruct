## Defining Modifiers

While defining a function the characters before the name (can be separated by a space, or just anything not alphanumeric) are split up and are modifiers.

#### EXAMPLE
{: .no_toc }
```
Input: [!func(cool)]

Name: func
Modifiers: !
```
